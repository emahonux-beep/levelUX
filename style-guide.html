02

Typography

Space Grotesk (bold) for headings — geometric with a slightly quirky lowercase g and a, distinctive without being loud. Karla for body — warm, humanist, highly legible at small sizes, and deliberately not Inter.

Display / H1 Bold headings
H2 Section titles
H3 Card & component headers
Body Karla reads comfortably at body sizes and holds up in long paragraphs.
Small / Caption Muted captions and metadata use the text-muted token.

This is a full paragraph set in Karla at the base size, so you can judge how it reads in real sentences rather than isolated words. It should feel calm and easy on the eyes, leaving Space Grotesk free to do the heavy lifting up in the headings.

04

How to use these tokens

1. Link tokens.css in your HTML. 2. Reference tokens by name in your own CSS — never hex codes directly.

/* in your stylesheet, after tokens.css is linked */
.card {
  background: var(--color-surface);
  color: var(--color-text);
  border: 1px solid var(--color-border);
  border-radius: var(--radius-lg);
}

.cta-button {
  background: var(--color-primary);
  color: var(--color-text-on-brand);
  font-family: var(--font-body);
}
