Advanced Views & Tools

schedule 5 min read
🎬 On-screen: title card, "Advanced Views & Tools"

You've got the fundamentals — components, variables, sectioning, navigation. This lesson rounds out the toolkit with the tools that come up less often, but matter a lot when you actually need them.

Layout guides and grids

🎬 On-screen: toggle a grid on and off over a frame

Applying a layout guide or grid to a frame gives you a consistent structure to align elements against, instead of eyeballing spacing frame by frame. This matters most on anything with repeating structure — a list, a dashboard, a form.

Slots

Slots are placeholder areas built to make swapping content in and out fast — useful when you've got a structure that stays fixed but the content inside it changes often, like a card template used across many pieces of content.

Masks and vectors

  • Masks clip one layer to the shape of another — a photo cropped to a circle, an image contained inside a custom shape. Standard technique, but easy to reach for once you know it's there.
  • Vectors let you go beyond the basic shape tool for custom icon work or illustrations that need points and paths you control directly.

Design panel vs. prototype panel

We mentioned this briefly in Lesson 5, but it's worth a closer look:

  • The design panel is where you control properties — size, fill, spacing, effects, variable bindings.
  • The prototype panel is where you wire up interactions — what happens when someone clicks, how one frame connects to the next, transition style and timing.
🎬 On-screen: switch between the two panels on the same frame to show what changes

Knowing which panel you need before you go looking saves real time. A lot of "where did that setting go" moments come down to being in the wrong panel entirely.

Exporting frames to PDF

For anything that needs to leave Figma entirely — a stakeholder handout, a printed reference, an offline review — you can export frames directly to PDF without needing a separate tool.

What's next

Lesson 9 covers branching — how to create one, when to use it, and how to merge it back into your main file once you're ready.
