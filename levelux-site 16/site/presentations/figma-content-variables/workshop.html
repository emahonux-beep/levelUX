Concept

What Are Content Variables?

Definition

Content variables are reusable text values stored in Figma that can be applied across your files and components.

Instead of typing the same copy into every layer, you define it once — and reference it everywhere.

💡 A single source of truth for your copy
Change it in one place → it updates everywhere it's used.
💬 Real examples
Button labels
"Add to Basket"
Product names
"Paracetamol 500mg"
Disclaimers
"Always read the label"
Form labels
"Enter your date of birth"
Error messages
"This field is required"
Nav labels
"My Account"
Benefits

Why Use Content Variables?

🔁
Consistency
Every component uses the exact same copy. No more "Add to cart" on one screen and "Add to basket" on another.
⚡
Efficiency
Update copy once and it flows through every file and component instantly. No manual find-and-replace.
📈
Scalability
Whether you have 10 strings or 10,000, the workflow is the same. Perfect as your design system grows.
🚀
Speed
Content updates no longer need a designer to open and edit files. Great for working alongside content teams.
🤝Single source of truth — designers, developers, and content teams all working from the same values.

How To

Creating Variables Manually

1Open your Figma file
2Open the Variables panel (Assets → Variables)
3+ New Collection — name it (e.g. "Pharmacy / Labels")
4+ Add Variable → choose type: String
5Name it clearly (e.g. btn/confirm-label)
6Enter the value (e.g. "Confirm Order")
7Repeat for each variable → then Publish
💡 Naming tip

Use a group/name format like section/variable-name

pharmacy/btn-confirm
pharmacy/error-required
✅ Best for
  • Small sets of variables
  • Quick one-offs
  • When you're just getting started

Each row becomes a variable, each column becomes a mode, and the file name becomes the collection. Use / in a name to make groups.
1Export the sheet as a CSV (name it after your collection)
2Variables panel → Import → run CSV to Variables → select your file
3Preview the table, import, then publish

Application

Applying Variables in Your File

Step by step
1Select a text layer in your file
2Open the Appearance section in the right-hand panel
3Click Appearance → the variable picker appears
4Browse or search for your variable
5Click to apply → the layer now references it
6It shows the value, not a hardcoded string
🧩 Within a component
  • Apply at the component level for best results
  • Instances can still be overridden if needed
  • Great for buttons, card text, tags, form fields
⚠️ Watch out

If the library doesn't appear in the picker:

Assets → Libraries → enable it for this file
